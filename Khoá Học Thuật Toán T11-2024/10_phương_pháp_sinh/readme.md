Cho trước N và K
Tìm số nhỏ nhất lớn hơn hoặc bằng N, sao cho N chia hết cho K
(1 ≤ N, K ≤ 10^18)

Ví dụ:
N = 10, K = 3, output = 12

N = 7, K = 3, output = 9

N = 10, K = 5, output = 10








Cho trước N và K
Tìm số M lớn nhất bé hơn hoặc bằng N, sao cho M chia hết cho K
(1 ≤ N, K ≤ 10^18)

Ví dụ:
N = 10, K = 3, output = 9

N = 7, K = 3, output = 6

N = 10, K = 5, output = 10

Số chia hết cho K là: K * x
(x là số tự nhiên)

(1 ≤ N ≤ 10^18)

Ví dụ:

N = 10, output = 9

N = 7, output = 4

N = 100, output = 100


Cho một một số nguyên dương N, tìm số chính phương nhỏ nhất lớn hơn N
(1 ≤ N ≤ 10^18)

Ví dụ:

N = 10, output = 16
