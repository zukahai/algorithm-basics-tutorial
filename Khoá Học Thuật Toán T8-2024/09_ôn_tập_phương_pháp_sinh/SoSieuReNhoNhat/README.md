## Số siêu lẻ nhỏ nhất

Số siêu lẻ nhỏ nhất là số mà có tất cả các chữ số đều lẻ.

Hãy tìm số siêu lẻ nhỏ nhất lớn hơn hoặc bằng **n**.

**Đầu vào:**

- Dòng đầu tiên chứa số nguyên **n** (1 ≤ **n** ≤ 10<sup>1000</sup>).

**Đầu ra:**

- Số siêu lẻ nhỏ nhất lớn hơn hoặc bằng **n**.

**Ví dụ:**

**Ví dụ 1:**

| Input | Output |
|:---:|:---:|
|20 | 31 |

**Ví dụ 2:**

| Input | Output |
|:---:|:---:|
|95 | 95 |
