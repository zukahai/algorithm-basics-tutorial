# Tổng các chữ số trong một số

**Đề bài:**

Cho một số tự nhiên **n**. Hãy tính tổng các chữ số của số đó.

**Đầu vào:**

- Số tự nhiên **n** (0 ≤ n ≤ 10<sup>18</sup>).

**Đầu ra:**

- Tổng các chữ số của số **n**.


**Ví dụ:**

| Đầu vào | Đầu ra |
| :--- | :--- |
| 1274 | 14 |

**Giải thích:**

- Số **n** = 1274. Tổng các chữ số của số **n** là: 1 + 2 + 7 + 4 = 14.
